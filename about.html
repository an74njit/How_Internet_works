About The Project

October 12th, 2020 by Alex V. Nunez

  • IS - 117

    This website is a project for the class of Introduction to Web Development (IS-117) in which we have applied the different concepts and programming languages for web design learned in class such as HTML, CSS, and Bootstrap.

    In this project, we will go over the main components of the Internet in an attempt to illustrate how it works.

    We will cover and explain very important concepts such as Domain names and DNS, HTTP protocols, The principal organizations in charge of standardization and regulation ICANN and W3C and the main languages developed to create web pages.

    We have used GitHub for the control version of the project and also to host it. Additionally, we have link a custom domain name to our web site got it at godaddy.com, and also we have hosted the project to have it available online on the NJIT Web Server (AFS)
