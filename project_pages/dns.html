Domain Name System (DNS)

October 12th, 2020 by Alex V. Nunez

  • DNS
    Definition

    The Domain Name System (DNS) is the Internetâ€™s system to translate domain names into numeric Internet Protocol (IP) addresses. In other words, is the phone book on the internet. Humans access information online using domain names, such as nytimes.com or sports.com. Web browsers interact using Internet Protocol (IP) addresses. DNS servers translate domain names to IP addresses so that browsers can load Internet resources.

    Each device connected to the Internet has a unique IP address that other devices can use to find the device. DNS servers eliminate the need for humans to memorize IP addresses such as 192.168.1.1, or new more complex alphanumeric IP addresses, such as 2400: cb00: 2048: 1:: c629: d7a2.


    How does DNS work?

    The DNS server converts a hostname (such as www.example.com) to an IP address supported by the computer (such as 192.168.19.1). An IP address is given to every device on the Internet, and that address will be needed to find the appropriate Internet device, just like a street address is used to find a private home. When a user wants to load a page, what the user types in their web browser example.com must be translated into an address that the computer can understand to locate the example.com web page.
